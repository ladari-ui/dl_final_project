💙 Mental Health Emotion Analyzer
The Mental Health Emotion Analyzer is an AI-powered web application that detects human emotions from text and speech input. It uses a fine-tuned Transformer model to analyze emotional tone and provide supportive feedback to promote mental well-being.  This tool helps users become more aware of their emotional state 
An AI-powered web application that detects human emotions from text and speech input using a fine-tuned Transformer model. The app provides emotional insights, confidence scores, visual graphs, and supportive wellness tips to promote mental well-being.

🌿 Features

✅ Analyze emotions from typed text
✅ Speak your feelings using microphone input
✅ Speech-to-text conversion
✅ Emotion prediction using a fine-tuned DistilBERT model
✅ Confidence score visualization (graph)
✅ Supportive wellness tips
✅ Clean and interactive Streamlit interface

🧠 Emotions Detected

The model predicts the following emotions:

😢 Sadness

😄 Joy

❤️ Love

😠 Anger
